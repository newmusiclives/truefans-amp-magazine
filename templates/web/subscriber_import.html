Upload CSV
Used when the CSV row doesn't specify an edition.
Instructions

CSV should have an email column. Optional columns: first_name, edition.

The importer recognises these column name variants:

  • email, Email, or EMAIL
  • first_name, First Name, name, or Name
  • edition or Edition (slug, e.g. fan, artist, industry)

Example CSV:

email,first_name,edition
alice@example.com,Alice,fan
bob@example.com,Bob,artist
